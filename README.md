# ragnexus

> Embed-anything knowledge engine with a gitnexus-style DX. Index a source once,
> query it from any agent via MCP — or ask it directly from the CLI.

See `PLAN.md` for the full roadmap. MVP scope: PDF service manuals, fully local,
zero API keys.

## Install

```bash
pip install -e .
```

## Usage

```bash
ragnexus --help
ragnexus list
```
